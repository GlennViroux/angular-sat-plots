GNSS satellite-station coverage tool

This is an animated and interactive visualization tool for GNSS satellite coverage by a network of IGS ground stations. I.e., the tool allows to configure any set of GNSS satellites and any set of IGS stations, visualizing which stations have which satellites in sight at any point in time.

The final program is hosted on this site! Check it out for yourself here.

These are some of the techonologies used in this project:

  • Javascript
  • Angular
  • D3JS
  • Python Flask

Website for a creative mind

A simple website build for displaying pieces of art, check it out here.

The website was build from scratch, using AngularJS, Firebase and AWS S3. The idea was to have a dynamic website displaying static images for the broad public, while these static images can be edited, removed or replaced in a dynamic way by the administrator only.

These are some of the techonologies used in this project:

  • AngularJS
  • Firebase
  • AWS S3
  • Typescript
  • CSS
  • HTML5

Algorithmic trading application

A backend algorithmic trading program, joined by a frontend application in order to monitor the results and control basic and more advanced aspects of the trading algorithm.

This program runs 100% automatically, buying and selling stocks on its own, but has added features to allow user input for various aspects of the algorithm.

These are some of the techonologies used in this project:

  • Python
  • Dart
  • Flutter
  • Python Flask
  • Yahoo Finance API
