Algorithmic trading application

A backend algorithmic trading program, joined by a frontend application in order to monitor the results and control basic and more advanced aspects of the trading algorithm.

This program runs 100% automatically, buying and selling stocks on its own, but has added features to allow user input for various aspects of the algorithm.

These are some of the techonologies used in this project:

  • Python
  • Dart
  • Flutter
  • Python Flask
  • Yahoo Finance API

GNSS satellite-station coverage simulator

This is an animated and interactive simulator for GNSS satellite coverage by a network of IGS ground stations. I.e., the simulator allows to configure any set of GNSS satellites and any set of IGS stations, visualizing which stations have which satellites in sight at any point in time.

The final program is hosted on this site! Check it out for yourself here.

These are some of the techonologies used in this project:

  • Javascript
  • Angular
  • D3JS
  • Python Flask
