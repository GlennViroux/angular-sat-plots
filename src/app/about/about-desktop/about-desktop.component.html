About me

Hello! I'm Glenn, a software engineer currently based in Madrid, Spain. I actually got my masters degree in mechanical/aerospace engineering, which originated from my interest in space science and everything related to it.

I studied mechanical/aerospace engineering in the Catholic University of Leuven, because of my general interest in understanding how things work. After graduating, I started working as a software engineer working in a project concerning the European Global Satellite Navigation System (GNSS), Galileo. Although my work in this project is highly focused on the backend part, I started being interested also by the frontend part of software applications. Fueled by this interest, I started working on some personal projects, involving both backend and frontend code. You can find an overview of these personal projects here.
